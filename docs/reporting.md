# Reporting

DefenseGraph provides fixed export templates for two audiences.

## Executive Report

Format: PDF

Includes:

- headline KPIs
- top risks
- scope summary
- tested-status summary
- short focus areas

Tone is concise and non-technical.

## Technical Report

Format: PDF

Includes:

- KPIs
- top risks
- current gap list
- theoretical vs real effect context
- confidence and tested state

Tone is more detailed and operational.

## Gap Export

Format: CSV

The CSV export contains the current gap-oriented technique list with effect, confidence, test status, and dependency notes.
